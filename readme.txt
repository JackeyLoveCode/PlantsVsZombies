已经实现的功能：
    1.绘制背景
    2.绘制顶部太阳数量栏和种子栏，太阳数量的显示，使用的是系统自带的字体
    3.种植太阳花，豌豆射手，坚果，植物图片随着鼠标移动而移动
    4.种下植物后，太阳的数量减少
    5.拾取太阳后，太阳的数量增加
    6.种下豌豆射手后，隔1秒生成一颗子弹，需要将豌豆射手的坐标传到子弹的构造器中
    7.种下太阳后，隔5秒生成一颗太阳
    8.生成太阳，生成子弹，生成僵尸，生成旗帜僵尸，都用到了自定义事件
    9.子弹与僵尸碰撞，子弹消失，僵尸生命值-2/次，当僵尸生命值到达某些临界点时，替换图片，但会存在一闪一闪的问题
    10.子弹与旗帜僵尸碰撞，子弹消失，旗帜僵尸-2/次，当旗帜僵尸生命值为0时,旗帜僵尸消失
    11.坚果和僵尸碰撞，改变了僵尸的运动形态，坚果每隔30秒生命值减1，当坚果生命值为0时，坚果消失，僵尸继续向前走
    12.豌豆射手与僵尸碰撞，改变了僵尸的运动形态，豌豆射手每隔10秒生命值减1，当豌豆射手的生命值为0时，豌豆射手消失，僵尸继续前行
    13.太阳花与僵尸碰撞，改变了僵尸的运动形态，太阳花每隔10秒生命值减1，当太阳花的生命值为0时，太阳花消失，僵尸继续前行
    14.坚果和旗帜僵尸碰撞，改变了旗帜僵尸的运动形态，坚果每隔30秒生命值减2，当坚果生命值为0时，坚果消失，僵尸继续向前走
    15.豌豆射手与旗帜僵尸碰撞，改变了旗帜僵尸的运动形态，豌豆射手每隔10秒生命值减2，当豌豆射手的生命值为0时，豌豆射手消失，僵尸继续前行
    16.太阳花与旗帜僵尸碰撞，改变了旗帜僵尸的运动形态，太阳花每隔10秒生命值减2，当太阳花的生命值为0时，太阳花消失，僵尸继续前行
    在僵尸碰到坚果时，要将坚果与僵尸列表绑定
    在坚果被吃完后，要将坚果与僵尸解绑
没有实现的功能：
    1.子弹与僵尸碰撞时没有效果
    2.坚果在被僵尸吃的过程中没有变化
    3.豌豆射手在被僵尸吃的过程中没有变化
    4.太阳花在被僵尸吃的过程中没有变化
    5.游戏结束功能
    6.游戏暂停
存在的BUG
    1.僵尸在啃食坚果，太阳花，豌豆射手时如果被子弹击中，僵尸形态有问题
    2.在同一块区域，可以种两个以上的植物
